收集一些游戏的 SDK、内存基址与偏移量数据，方便查阅和对比。
不定期更新，随缘维护。

## 文件说明

每个游戏文件夹内按类型分类存放：

- **cs/** — C# 源码
- **hpp/** — C++ 头文件
- **json/** — JSON 格式数据
- **rs/** — Rust 源码
- **zig/** — Zig 源码
- **dll/** — 虚拟 DLL
- **CppSDK/** — C++ SDK
- **Dumpspace/** — 分类 Dump 数据
- **IDAMappings/** — IDA 映射文件
- **Mappings/** — UE 映射文件
